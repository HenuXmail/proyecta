Bienvenido a la pagina de github del proyecto "Proyecta".

proyecta es un sistema de estadísticas el cual realiza proyecciones de ventas en base a una serie de parámetros.

Orientado al sector retail, puede ser utilizado por diversas áreas de la empresa, sobre todo para quienes deben tomar las decisiones de ventas de los productos

Instalación y configuración

Luego de Clonar el repositorio en tu maquina, debes seguir los siguientes pasos:

Cambiar la ruta de los templates de Django, Django usa rutas absolutas y no relativas, por ende debes colocar la que corresponda a tu instalacion.

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages
